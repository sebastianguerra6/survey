# Ejemplos de Preguntas del Banco de Evaluación

Este documento contiene ejemplos de preguntas organizadas por categorías que se pueden usar en el sistema de evaluación.

## Categorías de Preguntas

### 1. Comprensión del Problema (3 preguntas)

- **¿El analista demostró comprensión adecuada del caso y sus requisitos?**
  - Penalización Graduado: 5.0
  - Penalización No Graduado: 10.0

- **¿Identificó correctamente las partes interesadas y sus necesidades?**
  - Penalización Graduado: 4.0
  - Penalización No Graduado: 8.0

- **¿Comprendió el contexto del negocio y las implicaciones del caso?**
  - Penalización Graduado: 6.0
  - Penalización No Graduado: 12.0

### 2. Análisis y Metodología (4 preguntas)

- **¿La solución propuesta es técnica y metodológicamente correcta?**
  - Penalización Graduado: 10.0
  - Penalización No Graduado: 15.0

- **¿Utilizó un enfoque estructurado y sistemático para el análisis?**
  - Penalización Graduado: 8.0
  - Penalización No Graduado: 12.0

- **¿Aplicó herramientas y técnicas apropiadas para el tipo de problema?**
  - Penalización Graduado: 7.0
  - Penalización No Graduado: 11.0

- **¿Realizó un análisis de causa raíz adecuado?**
  - Penalización Graduado: 9.0
  - Penalización No Graduado: 14.0

### 3. Completitud y Profundidad (4 preguntas)

### 4. Comunicación y Presentación (4 preguntas)

## Notas

- Las penalizaciones para "No Graduado" son típicamente 1.5-2x las de "Graduado"
- Las preguntas de ética y cumplimiento tienen las penalizaciones más altas
- Las preguntas de comunicación tienen penalizaciones más bajas
- Los prefills se pueden configurar desde la ventana de administración de preguntas
